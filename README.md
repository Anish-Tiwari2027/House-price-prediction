# 🏠 House Price Prediction using KNN  

This project predicts house prices using **K-Nearest Neighbors (KNN)** algorithm.

It takes input like:
- Area (in square feet)
- Number of Bedrooms
- Age of the House  

and predicts the estimated house price 💰.

---

## ⚙️ Tech Stack
- Python 🐍  
- Jupyter Notebook  
- Flask (Backend API)  
- Scikit-learn (for KNN model)  
- HTML/CSS/JS (optional frontend)
